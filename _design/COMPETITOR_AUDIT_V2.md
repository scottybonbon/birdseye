## Summary: The Core Strategic Shift

**Current Birdseye positioning:** "24/7 remote guarding service for yards"  
**Competitor positioning:**
- Outpost: Infrastructure scale + cost reduction (70% savings)
- Terminal: AI-native system transformation (YOS™)
- LVT: Technical omniscience + vertical specialization
- Flock: Ecosystem + community impact + policy engagement

**Birdseye's rebuild should move toward:**
1. **System language** (not just service): "Intelligent Yard Ops Network" positioning
2. **Multi-stakeholder narrative:** Facility Managers, Security Directors, Compliance
3. **Quantified impact:** Not "24/7 guarding" but "40% faster response, 100% coverage"
4. **Technical authority:** Custom illustrations, operational tone, thought leadership
5. **Operational seriousness:** No hype videos; text, data, and diagrams
6. **Industry vertical focus:** Adapt messaging per yard type (Logistics, Ports, Automotive, etc.)

This bridges Birdseye's human expertise (remote operators) with Flock/LVT's technical credibility, while matching Outpost's operational gravitas.

**Design System Guardrails**

- **Primary Color (Navy #0F1C2E):** All body text, headers, navigation, primary UI
- **Secondary Color (Cream #F4EDE4):** Backgrounds, card fills, typography on dark
- **Accent Color (Electric #2E4BFF):** CTAs, highlights, animated elements, stat callouts
- **Typography:**
  - **Headings:** Instrument Serif Italic (h1, h2, large callouts)
  - **Body:** System font or serif (readability first)
  - **Data / Operational:** IBM Plex Mono (stats, metrics, datelines)
  - **Quotes/Accents:** Instrument Serif Italic
- **Illustration Style:** Custom schematics (navy lines, electric accents); no stock or generic
- **Photography:** Real-world facility shots, diverse operators, deployment scenarios; not posed corporate headshots
- **Video:** Minimal or none on homepage; use for product demos or company culture (separate from hero)
- **Testimonials:** Text-only, unattributed or minimal attribution, Instrument Serif Italic
- **Spacing:** Generous whitespace; navy + cream + electric need breathing room to feel premium
